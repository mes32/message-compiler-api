Message Compiler API

About

This is a simple webserver that composes customized messages.

Routes

  • GET /company ... Returns JSON array of all companies
  • GET /guest ... Returns JSON array of all guests
  • GET /template ... Returns JSON array of all available message templates
  • GET /message ... Requires query params: template, company, and guest providing IDs. Returns a message based on that template and inserting data from guest and company.
  • POST /message ... Takes a JSON object with companyID, guestID, template. Note: that template is a custom message template string and not a template ID.
